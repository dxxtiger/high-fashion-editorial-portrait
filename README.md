# High Fashion Editorial Portrait

A Codex skill for realistic adult high-fashion editorial portrait direction and image generation.

This skill understands Chinese, English, and Japanese styling instructions. It is designed for high-end magazine-style portrait imagery, including 4:3 editorial layouts, centered hero visuals, side detail panels, mature adult fashion allure, no-inner-layer couture styling, and prompt-only output when explicitly requested.

## Use

Install or copy this folder into your Codex skills directory:

```text
C:\Users\<you>\.codex\skills\high-fashion-editorial-portrait
```

Then restart Codex and invoke:

```text
$high-fashion-editorial-portrait
```

## Example

```text
$high-fashion-editorial-portrait 生成一张 4:3 横向高端时尚杂志版面：25岁成人模特，都市夜景，黑色廓形大衣，主视觉居中，右侧三张小图展示侧影、面料细节和步态变化。
```
